# Beer Study Analysis

## Problem
  To determine how satisfied female customers are with certain beer flavors.
  To determine which age group consumes more beer than the others.
  To identify which location has the highest beer consumption.

## Dataset
  ChatGPT generated: Beer Survey Sample Dataset

## Tools
  Excel, Tableau Public

## Process
  Cleaned dataset (handled missing values, standardized fields)
  Created pivot tables for analysis
  Built dashboard in Tableau

## Key Insights
  The most favorite beer flavor for females is IPA
  Pilsner is the most bought beer and gained most satisfactory as well
  Quezon City has the most beer consumption among all location
